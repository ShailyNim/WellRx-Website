Well Construction · Liner Accessories

Seal it, float it, land it, tie it back — one string, no surprises.

The supporting string — seals at the top, floats and collars at the bottom, and the tools that run and release it — designed and manufactured together as one integrated system.

The benefit to the operator is simple: no stitching together mismatched imports, no interface guesswork, one package that runs, seals and lands predictably. Flow-assisted sealing controls annular gas migration after cementing; drillable floats and collars mean fast, trouble-free drill-out; and running tools release cleanly even on long, heavy, extended-reach strings — so nothing is left downhole and nothing needs a second trip.

Liner Top Packers

HF-1

High-flow liner top packer, compression-set via TBR

Flow-assisted element, large by-pass, no hydraulics; secondary seal controlling annular gas migration after cementing.

HF-SG

High-flow Super Grip top packer with hold-down slips

Slips + lock ring lock applied force; runs standalone or with WellRx hangers for set-on-bottom / scab-liner pack-off and gas-migration control.

Tie-Back Equipment

AYAT

Polished bore receptacle (PBR)

Honed ID seal bore, shields running tool, 6–20 ft; tie-in point for seal mandrel / tie-back packer and liner-top packer extension.

AVDO

Setting sleeves, run above the hanger

Carries liner load, enables running-tool engagement & rotation; runs with any WellRx liner hanger.

VIPAM

Tie-back seal mandrel

Stacked seals, full-opening ID = liner ID; economical liner-top isolation in vertical wells, run liner to surface + cement.

VISAM

Tie-back mechanical packer, weight-set

3-element system seals into existing PBR; ties back casing to surface when existing casing leaks or is damaged.
